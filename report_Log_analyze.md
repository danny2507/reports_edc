# Self-Canonicalization Metrics Report

## Metric Definitions

- `n_top1_canon_relations`: number of distinct canonical relation labels that appeared as Top-1 after SC.
- `n_unstable_raw_relations`: number of distinct raw relation labels whose occurrences mapped to more than one Top-1 canonical label.
- `n_ambiguous`: number of scored triplet-level decisions where `Top1 score - Top2 score <= 0.05`.
- `n_raw_relations`: number of distinct raw relation labels extracted before canonicalization.
- `n_triplets`: number of extracted triplets analyzed for the case.

## Column Glossary

### Summary Tables

- `iter`: iteration folder that the rows came from, for example `iter0`.
- `case`: SC variant being analyzed, for example `case6_weighted`.
- `n_triplets`: total number of extracted triplets analyzed in this case.
- `n_raw_relations`: number of distinct raw relation strings before canonicalization.
- `n_top1_canon_relations`: number of distinct canonical relation labels selected as Top-1.
- `n_unstable_raw_relations`: number of raw relation strings that map to more than one Top-1 canonical label.
- `mapping_stability_share`: share of raw relation strings that are stable, computed as `1 - n_unstable_raw_relations / n_raw_relations`.
- `n_scored_gaps`: number of triplet decisions where both Top-1 and Top-2 scores were available.
- `mean_gap`: average `top1_score - top2_score` across scored triplet decisions.
- `median_gap`: median `top1_score - top2_score`.
- `p10_gap`: 10th percentile gap. Lower values indicate the weakest confidence region.
- `n_ambiguous`: number of scored triplet decisions where `gap <= 0.05`.
- `ambiguous_share`: `n_ambiguous / n_scored_gaps`.

### Mapping Diversity Tables

- `raw_relation`: raw predicate extracted from the text/OIE triplet before SC.
- `raw_general_definition`: SD2a general definition for this exact raw triplet when available.
- `raw_general_definitions`: unique SD2a general definitions observed for this raw relation group.
- `raw_general_relation_type`: SD2a relation kind label, such as `attribute` or `event`, when available.
- `raw_detailed_definition`: SD2b detailed typed definition for this raw relation in the item.
- `raw_detailed_definitions`: unique SD2b detailed typed definitions observed for this raw relation group.
- `raw_head_type`: SD2b inferred head/entity type for the raw relation.
- `raw_tail_type`: SD2b inferred tail/entity type for the raw relation.
- `raw_head_types`: unique SD2b head types observed for this raw relation group.
- `raw_tail_types`: unique SD2b tail types observed for this raw relation group.
- `raw_detailed_relation_type`: SD2b relation kind label, such as `attribute` or `event`, when available.
- `n_occurrences`: number of triplet occurrences with this raw relation.
- `n_unique_top1_canon`: number of distinct Top-1 canonical labels assigned to this raw relation.
- `top_canon`: most frequent Top-1 canonical label for this raw relation.
- `top_canon_definition`: schema definition of `top_canon` from `canon_schema_<case>.json`.
- `top_canon_count`: number of occurrences that mapped to `top_canon`.
- `top_canon_share`: `top_canon_count / n_occurrences`. Higher is more stable.
- `entropy_bits`: entropy of the raw-to-canon distribution. Higher means more spread across canonical labels.
- `canon_distribution`: full count distribution of Top-1 canonical labels for this raw relation.

### Confidence Gap Tables

- `n_scored`: number of occurrences for this raw relation with both Top-1 and Top-2 scores.
- `n_missing_score`: number of occurrences where a gap could not be computed from the SC log.
- `min_gap`: smallest observed `top1_score - top2_score` for this raw relation.
- `max_gap`: largest observed `top1_score - top2_score` for this raw relation.

### Reverse Mapping Tables

- `top1_canon`: canonical label selected as Top-1.
- `top1_canon_definition`: schema definition for `top1_canon` from `canon_schema_<case>.json`.
- `top1_canon_head_type`: schema head type for `top1_canon`.
- `top1_canon_tail_type`: schema tail type for `top1_canon`.
- `n_unique_raw`: number of distinct raw relations that mapped into this canonical label.
- `top_raw`: most frequent raw relation mapped into this canonical label.
- `top_raw_general_definitions`: SD2a definitions observed for `top_raw`.
- `top_raw_detailed_definitions`: SD2b definitions observed for `top_raw`.
- `top_raw_count`: number of occurrences from `top_raw`.
- `top_raw_share`: `top_raw_count / n_occurrences` for this canonical label.
- `raw_distribution`: full count distribution of raw relations mapped into this canonical label.

### Triplet Decision Tables

- `item_id`: input item index, matching `idx_<item_id>.sc.log`.
- `triplet_idx`: position of the triplet inside that item.
- `subject`: triplet subject.
- `object`: triplet object.
- `top2_canon`: second-ranked canonical relation candidate.
- `top2_canon_definition`: schema definition for `top2_canon` from `canon_schema_<case>.json`.
- `top2_canon_head_type`: schema head type for `top2_canon`.
- `top2_canon_tail_type`: schema tail type for `top2_canon`.
- `top1_score`: retrieval/similarity score for the Top-1 canonical label.
- `top2_score`: retrieval/similarity score for the Top-2 canonical label.
- `gap`: `top1_score - top2_score`. Smaller values mean more ambiguity.
